在用Putty访问Ubuntu系统时，vi编辑器里面的方向键有可能打出ABCD字母。解决方法是：

1、putty中设置： Terminal—>Features—>勾选 Disable application cursor keys mode

2、Ubuntu中配置：

打开跟目录下的 ~/.vimrc 文件：

写入：

set nocp

set backspace=2

保存退出。
