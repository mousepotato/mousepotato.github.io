Mac OS X下可以很方便的进行截屏。主要有3个快捷键：

- `Shift + Command + 3`:进行全屏截屏
- `Shift + Command + 4`:选定区域进行截屏，并保持成预先设定的图片格式
- `Shift + Ctrl + Command + 4`:选定区域进行截屏，但是只保存在内存中，方便在Word，Keynote中编辑时直接进行粘贴。
